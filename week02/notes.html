Programming Basics

  • Variable: Are written using "const" when variables will not be reassigned and "let" when variable might be reassigned.

Arrays, Logic, and Loops

  • Arrays[]: Ordered list of values

    • Example: const heros = [];
    Adding values: heroes[0] = Batman; heros=['Spiderman', 'Batman', 'Superman'];
    delete: Use to remove an item from an array. Eg. delete heroes[1]

  • Logics: "if" and "else" statements
  • Loops: Repeats a piece of code over and over again according to certain conditions.

    • Types: While loop and For loops
