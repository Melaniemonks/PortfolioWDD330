Forms

    This is a common method used to interact with web pages. It is made up of the form element which contains controls such as: input fields, select menus and buttons.
    • When a form is submitted, the information is sent to a back end language
    • The default type of input field istext, which is used for entering a short piece of text, such as a username.
    • input type='password'is used to enter passwords or secret information.
    • Button has a default type "submit

Object Oriented Programming

    This is used to model the representation of objects in the real world. There are three types: Encapsulation, polymorphism and inheritance.
    • Encapsulation: In this case, the inner workings are kept hidden inside the object and only the essential functionality are exposed.
    • polymorphism: This means that various objects can share the same methods.
    • Inheritance: This takes an object that already exists and inherit all the properties and methods
    • Clasess: Classes are used to define the blueprint.

Modular JavaScript

    A module is a self-contained piece of code that provides functions and methods that can then be used in other files and by other modules.
