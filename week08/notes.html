Transforms

Translation
  • Translation functions allows us to move elements left, right, up, or down. It is similar to the behavior of "position: relative;"

Canvas, SVG, Drag and Drop

  • This is the API we use to draw anything we can imagine, all through JS. It helps improve the performance of our websites
  • Fetch API is used to request and send data asynchronously across a network.
Canvas
  • The canvas elements takes both a width and height attribute
  • All drawings are made using a canvas JavaScript API
  • The canvas element is then stored in a variable
  • A context variable is also created. Eg var context = canvas.getContext("2d" )
  • For brush color use "strokeStyle"
Canvas vs SVG
  • They both are used to draw custom shapes
  • Canvas allows for pixel manipulation
  • canvas operates in immediate nodes
  • There is no acess to what is drawn on the canvas via the DOM
  • SVG is accessible via the DOM thus causing cancas to be much faster
  • SVG has Raphael Library, Inkscape
Drag and Drop
  • API helps specify is elements are dragable and what happens when they are dragged
  • Set the "draggable" attribute on any HTML element that you like to drag
  • Add event listner for any dragstart event
  • Add event listner for the dragover and drop event
